loopmodel.get_loop_model_filename() -- get the model PDB name

get_loop_model_filename(sequence, id1, id2, file_ext)
This routine returns the PDB file name of each generated loop model, usually of the form 'foo.BL000X000Y.pdb'. You can redefine this routine if you don't like the standard naming scheme. See also automodel.get_model_filename().
