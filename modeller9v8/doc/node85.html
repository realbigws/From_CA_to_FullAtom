loopmodel.select_loop_atoms() -- select region for loop optimization

select_loop_atoms()
By default, this selects all atoms near gaps in the alignment for loop optimization. You should redefine this routine if you do not have an alignment, or you wish to set a different region for loop optimization. See section 2.3 for an example.
