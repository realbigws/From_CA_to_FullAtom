energy_data.dynamic_lennard -- calculate Lennard-Jones restraints

If set to True, dynamic Lennard-Jones restraints are calculated, using equation A.89. The default value is False.

Lennard-Jones restraints require the model topology to first be generated with model.generate_topology().
