automodel.generate_method -- control initial model generation

This is used to build the initial model. It is usually set to generate.transfer_xyz, which builds the model based on the templates, but you can also set it to generate.generate_xyz to build it purely from the internal coordinates, or to generate.read_xyz to read it from a file (see section 2.2.9 for the easiest way to do this).
