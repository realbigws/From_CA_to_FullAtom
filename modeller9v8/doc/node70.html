automodel.special_patches() -- add additional patches to the topology

special_patches(aln)
This routine, which is usually empty, can be redefined by the user to make additional changes to the model topology (for example, to add user-defined disulfides, to renumber the residues, or to rename the chains). See Section 2.2.7 for an example.
