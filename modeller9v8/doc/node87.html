loopmodel.user_after_single_loop_model() -- analyze or refine each loop model

user_after_single_loop_model()
This is the analog of automodel.user_after_single_model(), and is called after building each loop model, before the output PDB file is written. It can be redefined by the user to perform analysis or additional refinement on each loop model.
